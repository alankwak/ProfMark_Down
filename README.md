# ProfMark_Down

ProfMark_Down is a custom Markdown parser built to improve functionality for Professor Mark Mahoney at Carthage College. The main changes from traditional Markdown focus on improving how code is displayed.

ProfMark_Down requires C++ 11.

ProfMark_Down is run by creating a forking the project and creating your own copy. Then use your IDE and run the program by using 2 files one that is your Markdown file and another that is your HTML file.

ProfMark_Dwon is run by running the main function. You can create your own tests to test for functionality you desire to implement.
